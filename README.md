## About The Project

There are 11 poses, including 2 standing, called Samastitih,  and 9 moves, called:

1. ekam 
2. dve
3. trīṇi
4. catvāri
5. pañca
6. ṣaṭ
7. sapta
8. aṣṭau
9. nava

In this design, we treat sapta=trini, astau=dve, nava=ekam

### Built With

* mediapipe
* matplotlib
* numpy
* opencv
